# Numerical Methods (Python)

Simple implementations of basic numerical methods in Python.

## Methods
- Bisection method
- False position method (Regula Falsi)

## Files
- `Bisection.py` – bisection algorithm
- `Falsi.py` – false position algorithm
- `StopFun.py` – stopping conditions
- `Functions.py` – example functions
- `main.py` – example usage

## How to run
```bash
python main.py
